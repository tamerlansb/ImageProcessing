Сагиндыков Тамерлан Бехмухамбетович
Задание 3: Анализ изображений
  обязательная часть задания [+]
  реализация алгоритма Хариса для детектирования углов [+]

Система: Microsoft Visual Studio 2015
ОС: Windows 7 64bit

Инструкция:
  Программа работает в консольном режиме. Параметры запуска:
  %programname% (input_image) (output_image) (command) [parameters...]
  %programname% (input_image_1) (input_image_2) (command)
 command:
	canny {sigma} {t1} {t2} детектор Канни с параметрами sigma и
		t1 и t2 в пределах от  0 до 1 ( эффективные пороги полученные эмпирически 0,1 и 0,04)
	harris {sigma} детекор Харриса 
Комментарии:
  1. Для работы программы необходим Microsoft .NET Framework 4.
